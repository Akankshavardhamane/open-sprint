# 🌌 Cursor Universe

Cursor Universe is a fun interactive web experiment where different particle effects react to the movement of your cursor.

Move your mouse across the screen to create beautiful visual effects and sounds. Each theme behaves differently and gives a unique experience.

## ✨ Features

- Cursor-based particle effects
- Multiple interactive themes:
  - ✨ Stars
  - 🔥 Fire
  - 🌊 Water
  - 🌌 Black Hole
  - ⚡ Lightning
- Click anywhere to trigger particle bursts
- Sound effects that match each theme
- Smooth animations using HTML Canvas

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Canvas API  
- Web Audio API  

## ▶ How to Run

1. Download or clone this repository
2. Open `index.html` in any web browser
3. Move your cursor around the screen to generate effects
4. Change the theme using the selector at the bottom

## 🎯 Project Idea

The goal of this project is to create a small creative web experience where visuals and sound react to user interaction. It explores how simple physics and particle effects can create engaging interactive art in the browser.
